# Prisma_Rewards

Benvenuto nel glorioso, non ufficiale e assolutamente indispensabile archivio delle **PrismaLab awards**.  
Qui manteniamo con orgoglio i modelli 3D che celebrano i membri del laboratorio durante le premiazioni.

Se stai cercando modelli seri, eleganti, professionali… questo non è il posto giusto.  
Se invece vuoi statuine improbabili e premi dall’alto valore simbolico (e basso valore strutturale), accomodati.

## Cosa trovi qui

- **Modelli 3D delle statuine**
  - `.stl`, `.obj`, `.step`: tutto ciò che serve per stamparle, modificarle o osservarle chiedendosi “perché l’abbiamo fatto?”
- **Anteprime e render**
  - Immagini dei modelli in tutta la loro gloria.
- **Consigli di stampa**
  - Suggerimenti per evitare che la statuina collassi su sé stessa come uno studente il giorno prima della consegna della tesi.
- **Varie ed eventuali**
  - File extra, versioni alternative, errori volontari.

## Contribuire

Questo è un progetto **comunitario e non ufficiale**, perciò ogni contributo è benvenuto:

1. Fai una fork.
2. Aggiungi la tua statuina, meme geometrico o versione alternativa.
3. Apri una Pull Request.
4. I revisori valuteranno il tuo contributo con rigore scientifico pari a zero, ma con entusiasmo massimo.
